14 天总行程

多伦多 ⇄ 南通 · 含云南 + 顺德
三兄弟同行 · Yao 婚礼 11/1（南通）· 两套方案任选
两个人从多伦多飞、一个本在南通。南通、顺德住朋友家 不花住宿费。 下面 方案 A（先婚礼后旅游）和 方案 B（先旅游后婚礼）整条路线、交通、费用都列好了。
国际机票建议订「上海进 / 广州（或香港）出」联程，省一大段折返。云南那 5 天的细节单独成页： 👉 云南 5 天详细攻略 ›
方案 A 先婚礼，后旅游 ★ 推荐 · 不走回头路
路线：上海 → 南通 → 丽江 → 大理 → 广州/顺德 → 回程
一路向南、几乎不折返，最后从广东出境最顺。交通费（不含国际机票）约 ¥1,680/人。
数字 = 城市先后顺序 · 虚线 = 走的路线（A 一路向南）
1✈️ 抵达中国Day 1–2
✈️ 多伦多 → 上海浦东 直飞约 13–15 小时；🚄 浦东 → 南通高铁 ¥77 · 约 1.5h。

落地上海，高铁直接到南通，先和家人会合、休整倒时差。

2🎉 南通 · 婚礼Day 2–4

11/1 Yao 婚礼 🎊，前后陪家人、走动走动。住南通家里 不花钱。

3🏔 云南 5 天Day 4–9
✈️ 南通/上海 → 丽江 飞机 约 ¥600 · 4h（高铁要 18h，不考虑）。

丽江 2 晚（古城 + 玉龙雪山）+ 大理 3 晚（洱海躺平）。详细每天安排见上方「云南 5 天详细攻略」。

4🍲 顺德 · 陪 SimonDay 9–12
✈️ 大理 → 广州 飞机 约 ¥900；🚆 广州 → 顺德 城际/地铁 ¥30 · 40min。

顺德 3 晚，陪 Simon、吃遍顺德双皮奶与鱼生。住顺德 不花钱。

5✈️ 返程Day 12–13

顺德 → 广州 / 香港 → 多伦多。建议联程机票从广州或香港直接回，不用再北上回上海。

方案 B 先旅游，后婚礼 婚礼当天状态最好
路线：上海 → 丽江 → 大理 → 南通 → 顺德 → 回程
先去云南把时差倒过来，婚礼当天精神最好；代价是云南→南通要折返、多一段长途，交通费约 ¥2,400/人。
数字 = 城市先后顺序 · 看得出 B 在云南和南通之间有一段折返
1✈️ 抵达 + 直飞云南Day 1–2
✈️ 多伦多 → 上海浦东（约 13–15h）→ 当天/次日 ✈️ 飞丽江 约 ¥600。

落地后直奔云南，先用度假节奏把时差和状态调整好。

2🏔 云南 5 天Day 2–7

丽江 2 晚 + 大理 3 晚，洱海边躺平、调好时差。详细见「云南 5 天详细攻略」。

3🎉 南通 · 婚礼Day 7–10
✈️ 大理 → 上海/南通 飞机 约 ¥1,000（往东北飞回，这段是折返）。

11/1 Yao 婚礼 🎊，状态满满地参加。住南通家里 不花钱。

4🍲 顺德 · 陪 SimonDay 10–13
✈️ 南通 → 广州 飞机 约 ¥700；🚆 广州 → 顺德 城际 ¥30。

顺德 3 晚，陪 Simon、吃美食。住顺德 不花钱。

5✈️ 返程Day 13–14

顺德 → 广州 / 香港 → 多伦多。

怎么选？

· 方案 A（推荐）：路线一路向南最顺、交通费更低（约 ¥1,680/人），最后从广州/香港直接回。唯一要忍的是落地隔天就办婚礼、可能带点时差。

· 方案 B：先在云南倒好时差、婚礼当天状态最好，但要折返、交通费约 ¥2,400/人（多约 ¥700）。

· 两案住宿：南通、顺德住朋友家不花钱；云南 5 天约 ¥3000–3500/人。国际机票另计。
